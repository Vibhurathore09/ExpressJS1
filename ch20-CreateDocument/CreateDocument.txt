save() -> It is used to save document by inseriting a new document into the database if document.isNew is true , or sends an updateOne operation only with the modifications to the databse , it does not replace the whole document in the latter case.

It returns undfined if used with callback or a Promise otherwise.


example with callback
studentDoc.save((err , result) => {
    if(err){
        log(err);
    }else{
        log(result);
    }
})

const result = await studentDoc.save()
log(result);

Mongoose validates modified paths before saving. If you set a field to an invalid value , Mongoose will throw an error when you tru to save () that document.

const result = await studentDoc.save({validateBeforeSave : false});
// Use above to check validation before saving
